This project transforms raw CSV data into executive-level insights to support financial monitoring, operational efficiency, and customer experience optimization.

---

# 📊 Project Overview

This project analyzes insurance policy and claims data to evaluate:

- Premium revenue performance  
- Coverage vs claim amount comparison  
- Claims distribution by status (Rejected, Settled, Pending)  
- Active vs inactive customer ratio  
- Policy type contribution  
- Customer feedback and satisfaction trends  

The goal is to provide decision-makers with a clear, data-driven view of operational and financial performance.
